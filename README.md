# Dynasty_ADV
本工程是游戏 《天地劫序传：幽城幻剑录》同人视觉小说游戏的脚本，仅限学习和玩家自娱自乐，严禁用于任何商业活动。

# 游戏说明
1. 本脚本需要结合游戏引擎和资源使用。
2. 右上角三个按钮依次是保存、读取、从头开始。

# 脚本语法
和 python 类似的语法，注意缩进只能用空格不能用制表符（即Tab）
## 注释
行首用 # 注释这一行
## setvar X=表达式
设置变量 X 的值为表达式的值

    例如：
        setvar 冰璃好感=1
        setvar 冰璃好感=冰璃好感+1

## say 人物 内容
让人物说出内容, 如果在 assets/content/speaker/ 文件夹中存在 [speaker].png，就会显示该角色头像

    例如：
        say 夏侯仪之母 「阿仪，起床啦！」

## say "内容"
旁白说出内容

    例如：
        say "得到100两银子"

### 进阶：
人物以及内容都支持用 {} 格式化，所以你可以写

    setvar 银子=100
    say 夏侯仪 现在有{银子}两！

    setvar 主角名="夏侯仪"
    say {主角名} 我是{主角名}

## bg 文件名
读取 assets/content/bg/ 下的文件作为背景

## video 文件名
播放 assets/content/video/ 下的视频文件

## music 文件名
播放 assets/content/audio/bgm 下的视频文件作为背景音乐

## label/ jump 标签名
在这句话所在位置设一个标签，然后可以通过 jump 跳转到这个标签

    例如：
        label 开始
        say 夏侯仪 我好像被困住了
        jump 开始

## choice
选择跳转，和 label 相搭配
具体语法:

    choice:
        "选项名1" ?> 表达式1 -> 标签名1
        "选项名2" ?> 表达式2 -> 标签名2
        ...

    对每个选项如果后面的表达式不满足就不会出现这个选项，如果出现这个选项就能在游戏里点击并跳到标签对应的地方

    例如：
        setvar 冰璃好感=0
        choice:
            "开头" -> 开头
            "兰州城" -> 兰州城
            "无人废屋" -> 无人废屋
            "迦夏之窟" ?> 冰璃好感==1 -> 迦夏之窟
    在游戏里就只会显示开头、兰州城、无人废屋三个选项
    
## if
条件判断

    例如：
    if 冰璃好感==1:
        say "冰璃好感+1, 现在冰璃好感为{冰璃好感}"
    else:
        say "现在冰璃好感为{冰璃好感}"
## 已知的问题和不足
1. 音乐文件不支持网络路径和映射网络驱动器
2. 只有一个存档能用
3. 现在的 choice 语句必须跳转且不能加别的逻辑，但有时候 choice 只是希望运行某些逻辑，还需改进
4. if 还不支持 elif
